Detecting and characterising an exoplanet using data from a light-curve

The aim of this project is to detect an exoplanet using the data from a light-curve (a series of brightness measurements taken at specific times) of the star WASP-5.

We see that the star has a periodic dip in brightness, indicating the possibility of an exoplanet (which has been detected and called WASP-5b) orbiting this star. To test this out, we implement the Box-Least-Squares (BLS) algorithm to first find the period of WASP-5b. The algorithm consists of a phase folding and a binning phase.

The phase folding shifts the data, so that each period start at the same point. We search through a range of periods to determine the best-fit period. In the binning phase, we group the folded data into bins according to the trial period, so that the noise is reduced. We then calculate $\Delta \chi ^{2} _j$ (a measure of the fit quality) and $\delta _j$ (the uncertainty), for the data points within each bin, and record the maximum value of $\Delta \chi ^{2} _j$ for each trial period, defined as $\Delta \chi ^{2}$. Lastly, the period with the highest value of $\Delta \chi ^{2}$ will give the best-fit period.

Below is the code that calculates these 2 statistics and returns the trial periods, $\Delta \chi ^{2}$, $\delta$ and the epoch, $t_0$ (defined as the mid-time of the transit signature).

#statistics for each data point
weight=1/np.square(flux_error)
f_hat = np.sum(flux*weight)/np.sum(weight)
f_tilde = flux-np.full(np.shape(flux), f_hat)
T=sum(weight)

def bls(trial_period):
        phi_i = np.mod(t-t[0], trial_period) + t[0]

        #binning phase
        #create bins
        expected_duration = 2/24
        bin_edges = np.arange(min(phi_i), max(phi_i), expected_duration)
        distance_from_centre = np.append(np.diff(bin_edges) / 2, (np.diff(bin_edges) / 2)[0])
        bin_centres = distance_from_centre + bin_edges 
        bin_indices=np.digitize(phi_i, bin_edges) 

        #calculate the statistics for data points that fall in each bin 
        S_j=[] 
        R_j=[]

        for j in range(1, len(bin_edges)+1):
            f_i=[] 
            w_i=[] 
            index = np.where(bin_indices==j)[0] 
            
            for i in index:
                f_i.append(f_tilde[i])
                w_i.append(weight[i])
            f_i=np.array(f_i)
            w_i=np.array(w_i)

            S_j.append(np.sum(f_i*w_i))
            R_j.append(np.sum(w_i))

        return np.array(S_j), np.array(R_j), np.array(bin_centres), np.array(phi_i)

def algorithm():
    """function that returns 4 arrays. The trial period, the maximum values of chi_{j}^2 for each trial period, and the depth and epoch associated with each chi """
    P=np.arange(1, 10, 0.01)  
    max_chi = [] 
    max_delta = [] 
    max_t_0 = [] 
    for i in P: 
        denominator = (bls(i)[1] * (T - bls(i)[1]))
        delta_j = bls(i)[0] * T / denominator
        chi_j = np.square(bls(i)[0]) * T / denominator

        index_max_chi = np.where(chi_j == max(chi_j))[0][0]

        max_t_0.append(bls(i)[2][index_max_chi])
        max_chi.append(chi_j[index_max_chi])
        max_delta.append(delta_j[index_max_chi])

    return P, max_chi, max_delta, max_t_0

P, max_chis, delta, t0 = algorithm()

However, the calculated values will be good enough for a first approximation.

To find a better period, we make a basic model of the flux of the star, and fit this model to the actual data. This model flux will depend on some parameters like the radius of the exoplanet and its orbital radius, allowing us to then calculate the actual values.

In[5]:
def F(p, z):
    def l(p, z):
        if 1 + p < z:
            return 0
        
        elif abs(1-p) < z <= 1+p:
            kappa_0 = np.acos(((p**2 + z**2 - 1) / (2 * p * z)))
            kappa_1 = np.acos(((1 - p**2 + z**2) / (2 * z)))
            sqrt = np.sqrt((4*z**2 - (1 + z**2 - p**2)**2) / 4)
            return (1/np.pi) * (p**2 * kappa_0 + kappa_1 - sqrt)
        
        elif z <= 1-p: 
            return p**2
        
        elif z<=p-1:
            return 1
        
    return 1 - l(p, z)


def model_lightcurve(time_stamps, orbital_period, epoch, orbital_separation, size_ratio):
    theta_i = 2*np.pi*(time_stamps - epoch) / orbital_period
    z = abs(orbital_separation * np.sin(theta_i))

    f=[]
    for i in z:
        f.append(F(size_ratio, i))

    
    ind = np.where(np.cos(theta_i)<0)[0]
    for i in ind:
        f[i]=1 

    return f

p = np.sqrt(abs(associated_delta))
